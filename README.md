🌐 Website Project (HTML & CSS)

This is a simple and responsive website built using HTML and CSS. The project demonstrates the fundamentals of front-end web development, including page structure, styling, and layout design.

Clean and modern UI
Responsive design (works on different screen sizes)
Structured layout using HTML
Styled components using CSS
Easy to understand and beginner-friendly code
🛠️ Technologies Used

CSS3 – for styling and layout design
